# Penggunaan Generative AI dalam Pengurusan Universiti

## Gambaran Keseluruhan

Generative AI ialah teknologi kecerdasan buatan yang mampu menjana kandungan baharu seperti teks, analisis, idea, dan ringkasan maklumat berdasarkan arahan yang diberikan oleh pengguna.

Dalam konteks universiti, Generative AI boleh digunakan sebagai **pembantu digital** untuk membantu staf mempercepatkan kerja, menjana idea baharu, dan meningkatkan kualiti komunikasi organisasi.

### Kegunaan utama Generative AI dalam pengurusan universiti

| Fungsi              | Kegunaan                                                  |
| ------------------- | --------------------------------------------------------- |
| Penjanaan idea      | Membantu menghasilkan cadangan strategi dan program       |
| Penulisan kandungan | Menulis artikel, laporan, ucapan dan kandungan komunikasi |
| Ringkasan maklumat  | Meringkaskan laporan, artikel atau dokumen panjang        |
| Terjemahan teks     | Menterjemah dokumen dengan cepat dan tepat                |


# Kerangka Penggunaan AI

Penggunaan Generative AI yang berkesan boleh difahami melalui kerangka berikut.

```
Kenal pasti tugasan
        ↓
Nyatakan konteks organisasi
        ↓
Tulis prompt yang jelas
        ↓
Semak hasil AI
        ↓
Sunting dan gunakan hasil tersebut
```

Kerangka ini membantu memastikan hasil yang dijana oleh AI lebih relevan dan berkualiti.


# Kerangka Prompt Engineering

Untuk mendapatkan hasil yang lebih baik daripada AI, gunakan struktur prompt berikut.

```
Tugasan
+ Konteks
+ Audiens
+ Format Output
```

### Contoh Prompt

```
Cadangkan strategi pembangunan pelajar universiti yang inovatif. 
Fokus kepada peranan Jabatan Timbalan Naib Canselor (Hal Ehwal Pelajar dan Alumni). 
Jawapan hendaklah sesuai untuk pembentangan kepada pengurusan universiti.
Senaraikan cadangan dalam bentuk poin.
```


# Modul 1: Analisis dan Idea Strategik

Generative AI boleh membantu menjana idea strategik untuk pembangunan pelajar dan penglibatan alumni.

### Contoh penggunaan

| Tujuan                   | Contoh Prompt                                                                                                         |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------- |
| Analisis visi organisasi | Terangkan peranan Jabatan Timbalan Naib Canselor (Hal Ehwal Pelajar dan Alumni) dalam pembangunan pelajar universiti. |
| Analisis SWOT            | Lakukan analisis SWOT bagi program pembangunan pelajar universiti.                                                    |
| Cadangan program         | Cadangkan lima program inovatif untuk meningkatkan penglibatan pelajar di kampus.                                     |
| Penglibatan alumni       | Cadangkan strategi untuk meningkatkan hubungan antara universiti dan alumni.                                          |
| Pengurusan risiko        | Huraikan risiko utama dalam pelaksanaan program pelajar dan cara mengatasinya.                                        |



# Modul 2: Penjanaan Kandungan Kreatif

Generative AI boleh membantu menghasilkan kandungan komunikasi yang menarik dan berkualiti tinggi.

### Contoh kandungan yang boleh dijana

| Jenis Kandungan  | Contoh Prompt                                                                              |
| ---------------- | ------------------------------------------------------------------------------------------ |
| Artikel          | Tulis artikel tentang kepentingan pembangunan kepimpinan pelajar dalam pendidikan tinggi.  |
| Blog             | Tulis blog tentang pengalaman pelajar menyertai program pembangunan kepimpinan universiti. |
| Ucapan           | Tulis ucapan motivasi kepada pelajar baharu semasa minggu orientasi universiti.            |
| Cerita inspirasi | Tulis cerita inspirasi tentang alumni universiti yang berjaya dalam kerjaya global.        |
| Surat rasmi      | Tulis surat rasmi menjemput alumni menghadiri majlis universiti.                           |

# Modul 3: Ringkasan Maklumat

AI boleh membantu meringkaskan dokumen yang panjang supaya mudah difahami.

### Contoh penggunaan

| Jenis Dokumen        | Contoh Prompt                                             |
| -------------------- | --------------------------------------------------------- |
| Artikel akademik     | Ringkaskan artikel ini dalam lima poin utama.             |
| Laporan penyelidikan | Ringkaskan laporan kajian ini dalam bentuk poin ringkas.  |
| Bab buku             | Ringkaskan bab buku ini dalam satu perenggan.             |
| Laporan tahunan      | Ringkaskan pencapaian utama organisasi dalam laporan ini. |
| Kajian kes           | Ringkaskan kajian kes ini dan nyatakan pengajaran utama.  |



# Modul 4: Terjemahan Teks

Generative AI boleh membantu menterjemah dokumen dengan pantas antara pelbagai bahasa.

### Contoh penggunaan

| Jenis Dokumen | Contoh Prompt                                                         |
| ------------- | --------------------------------------------------------------------- |
| Surat rasmi   | Terjemahkan surat ini daripada Bahasa Melayu ke Bahasa Inggeris.      |
| Artikel       | Terjemahkan artikel ini ke Bahasa Melayu dengan gaya bahasa akademik. |
| Brosur        | Terjemahkan kandungan brosur universiti ke Bahasa Inggeris.           |
| Pengumuman    | Terjemahkan pengumuman universiti kepada pelajar antarabangsa.        |

# Perbandingan Penggunaan AI

| Tanpa AI                                | Dengan AI                                   |
| --------------------------------------- | ------------------------------------------- |
| Mengambil masa lama untuk menyusun idea | AI boleh menjana draf awal dengan cepat     |
| Penulisan memerlukan masa yang panjang  | AI membantu mempercepatkan proses penulisan |
| Analisis perlu dilakukan secara manual  | AI membantu menghasilkan cadangan analisis  |


# Latihan Bengkel

## Latihan 1 – Menjana Idea

Gunakan AI untuk menghasilkan cadangan program pembangunan pelajar.

```
Cadangkan lima program inovatif yang boleh meningkatkan pembangunan kepimpinan pelajar universiti.
Terangkan objektif dan impak setiap program.
```

## Latihan 2 – Penulisan Kandungan

```
Tulis artikel ringkas tentang kepentingan pembangunan kepimpinan pelajar universiti.
```


## Latihan 3 – Ringkasan Maklumat

```
Ringkaskan teks berikut dalam lima poin utama supaya mudah difahami oleh staf universiti.
```


# Prinsip Penggunaan AI Secara Bertanggungjawab

Dalam penggunaan Generative AI, staf universiti perlu mengambil kira beberapa prinsip penting:

1. Semak ketepatan maklumat yang dijana oleh AI
2. Pastikan maklumat sensitif tidak didedahkan kepada sistem AI
3. Gunakan AI sebagai alat sokongan, bukan pengganti kepakaran manusia
4. Sentiasa melakukan semakan dan penyuntingan terhadap hasil AI


# Kesimpulan

Generative AI merupakan alat yang sangat berpotensi untuk meningkatkan produktiviti kerja dalam organisasi universiti. Dengan penggunaan prompt yang jelas dan struktur yang betul, staf boleh menjana idea, menghasilkan kandungan berkualiti, serta menganalisis maklumat dengan lebih cepat dan berkesan.

AI bukan bertujuan menggantikan manusia, tetapi berfungsi sebagai **pembantu digital yang membantu mempercepatkan kerja dan meningkatkan kualiti hasil kerja**.
